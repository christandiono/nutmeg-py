nutmeg.stats.stats_utils.map_t_py(t, pvals, dp)¶

In a set M of maximal statistic values, we can define a cumulative distribution function as follows:

CDF[t] –> n*(dp) := order{ tm | (tm in M) and tm < t } / order{M}

Given quantized p values in the set { n*(dp) } for some n in [0,1/dp) and corresponding test values, his method attempts to roughly invert the relationship of the CDF.
